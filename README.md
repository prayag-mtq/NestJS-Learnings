## Introduction to NestJS

- Nest (NestJS) is a framework for building efficient, scalable Node.js server-side applications
- Under the hood Nest makes use of Robust HTTP server frameworks like Express.
- It's architecture is heavily inspired by Angular.

### Installation

```bash
  npm i -g @nest/cli
  nest new project-name
```

### Booting up the project

```bash

  cd project-name
  npm run start
```
